# Generador de Contraseñas Seguras

## Integrante

* (Cuascota Gualacata Jhon Amilcar)


## Objetivo del sistema

Desarrollar un programa que genere contraseñas seguras de manera automática utilizando Python, con el fin de mejorar la seguridad digital.

## Descripción de funcionalidades

* Generación de contraseñas aleatorias
* Uso de letras, números y símbolos
* Configuración de longitud de contraseña
* Visualización del resultado en pantalla

## Tecnologías utilizadas

* Python
* Visual Studio Code
